Currency Exchange API

This Currency Exchange API is REST-ful and supports CORS, so it can also be used in web browsers.

Routes

GET / (text/html)

Returns an HTML document with API routes and usage.

GET /convert (application/json)

Returns a JSON object with the converted currency, using the latest exchange target rates published by the European Central Bank.

Query parameters:

  • from: 3-letter ISO-4217 code for the base currency.
  • to: 3-letter ISO-4217 code for the quote currency.
  • amount: Amount to be converted, in base currency. Use a dot (.) as decimal separator, grouping separators are not allowed.
Response:
  • amount (string): The converted amount as a string, with 4 decimal places.
  • currency (string): 3-letter ISO-4217 code for the target currency used for the conversion.
  • rate (string): The effective conversion rate applied, with 8 decimal places
  • date (string): The date of the rates returned from the ECB.
